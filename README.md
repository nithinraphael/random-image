# Random-image

A minimal Flutter app that fetches a random image from an API, displays it in a polished UI, and caches previously loaded images for instant reloads.

## Demo

https://github.com/user-attachments/assets/1f555382-e512-42bf-8d89-eada51b8d28c

---

## Features

* **Fetches random images** from the API
* **Caches images** to prevent repeat downloads
* **Handles errors safely** with fpdart
* **Uses Signals** for simple global state
* **Smooth UI** with fade-in and adaptive background

---

## Running the App

```sh
flutter pub get
flutter run
```
